# AVIT 3257 · Brochure web

Brochure web responsive e interactivo del proyecto **AVIT 3257 — Apartamentos**, un desarrollo de **Foquier Desarrollos** sobre Av. Italia, Montevideo.

## Contenido

```
index.html          → la página (todo el CSS y JS va embebido)
assets/img/         → renders, planos e imágenes del proyecto
```

## Cómo publicarlo en GitHub Pages

1. Crear un repositorio nuevo en GitHub (por ej. `avit-3257`).
2. Subir `index.html` y la carpeta `assets/` a la raíz del repo.
3. En el repo ir a **Settings → Pages**.
4. En **Source** elegir la rama `main` y la carpeta `/ (root)`. Guardar.
5. En unos minutos el sitio queda online en:
   `https://TU-USUARIO.github.io/avit-3257/`

También podés abrir `index.html` directamente en el navegador para verlo localmente.

## Características

- Diseño 100% fiel al brochure original (colores, tipografía, logos y renders).
- Totalmente responsive (escritorio, tablet y celular).
- Navegación con menú fijo y scroll suave entre secciones.
- Mapa de ubicación interactivo (Google Maps).
- Galería con lightbox y planos ampliables al hacer clic.
- Botón de contacto directo por email.
